# Navigation Patterns Identified

## 1. Horizontal Top Navigation
- Used by Amazon, Flipkart, BBC.
- Best for desktop with multiple categories.
- Pros: easy scanning; good for mega menus.

## 2. Hamburger + Off-canvas Menu
- Common for mobile.
- Pros: saves space.
- Cons: extra tap required.

## 3. Sticky Header Navigation
- Search bar or header stays visible while scrolling.
- Seen in Amazon, YouTube.
- Pros: improves usability.

## 4. Bottom Navigation (Mobile)
- Used by Airbnb & YouTube apps.
- Best when user performs repeated actions.
- Pros: thumb-friendly.

## 5. Mega Menu
- Large dropdown with multi-level categories.
- Used by e-commerce sites like Amazon.
- Pros: handles large data without clutter.

## 6. Minimalist Navigation
- Used by Medium.
- Focuses more on content.
- Pros: very clean UI.
